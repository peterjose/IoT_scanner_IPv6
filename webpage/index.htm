IoT-Speaking Hosts Discovery Service

On this website we present the results of the ongoing active measurement to discover IoT-speaking hosts.

This work is conducted as a part of Master Thesis Discovering and Characterising IoT-Speaking Hosts in IPv6 Internet. The active measurement is done on IPv6 internet, using the IPv6 hitlist service.

Data Request

In order to facilitate further research, we share raw measurement results with fellow researchers. The data is sensitive, so you must send a quick registration email in order to gain access. Detailed information about the work department, the purpose of the data usage, and the data content should be included in the request.

Tools

Open Port Scanning Tool

We have used Zmap tool from TUM which supports IPv6, along with our few custom probes for doing the open port scanning.

Application-Layer Network Scanner

We extend the Zgrab2 tool to support popular IoT protocols like AMQP, XMPP, CoAP, MQTT and OPC UA. We also added support for DTLS to extend the scanner for scanning the secured version of CoAP. The code is available at link
